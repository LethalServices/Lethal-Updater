# About 
Allows you to update lethal without going to the server to download via our discord bot. 

## How To Update:

- Run via cmd.exe/Windows Terminal & input your license key. 

```cs
python Updater.py
```

## Requirements:
```
- tqdm
- getpass
- zipfile
- sqlite3
- colorama 
```

## Developer Information:
- Python: 3.10.8
